# LLM-Templates

A concise collection of reusable prompt templates, refinement commands, and workflows for working with large language models. Files are short, focused, and designed to be copied into an LLM interface (e.g., chat.openai.com) for rapid iteration.

Repository structure
- /prompts
  - 004-prompting-basics.md — Overview & how-to
  - ideation.md — Brainstorming, MVP scoping, assumption mapping
  - biz-dev.md — Cold emails, business model canvas, investor pitch outlines
  - research.md — Market landscape, competitor analysis, discovery questions
  - creative.md — One-pagers, LinkedIn posts, presentation outlines
  - refinement.md — Quick edit and tone commands
  - workflow.md — AI-assisted career acceleration workflow
- LICENSE — MIT
- .gitignore

How to use
1. Open a file in /prompts and copy a template.
2. Replace bracketed fields (e.g., [INDUSTRY]) with your specifics.
3. Paste into an LLM and run. Use the evaluation checks in each file (Clarity, Usability, Completeness) to assess results.

Contributing
- Submit improvements or example runs via pull requests.
- Remove or avoid adding any private or sensitive data.

License
MIT
